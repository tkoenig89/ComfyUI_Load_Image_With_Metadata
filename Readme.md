# Load Image with metadata

A custom node for comfy ui to read generation data from images (only png for now).

* positive prompt (STRING)
* negative prompt (STRING)
* seed (INT)
* size (STRING: eg. 512x512)
* cfg (FLOAT)
* steps (INT)
